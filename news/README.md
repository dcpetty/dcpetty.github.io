# [News](https://dcpetty.github.io/news/)

This tool was written with [PyScript]() &mdash; a way of mixing [Python](https://docs.python.org/3/) with [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) &mdash; for those of us comfortable with things [Pythonic](https://realpython.com/ref/glossary/pythonic/).

This tool creates a tracker-free, clickable, short [Google Search](https://www.google.com/search) link to search [Google News](https://www.google.com/news). The link updates as you enter your query.
